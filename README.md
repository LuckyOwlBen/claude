# How to turn a computer with a graphics card into claude (I didn't come up with this but I want to remember)

1 Install Ollama
2 Install the gemma4-claude file into a new folder in ~/.ollama
3 Install claude plugin to vs code
4 Use settings.json included here to overwrite or update the settings.json in the ~/.claude folder.
5 run the below commands in this repo assuming you have python3 installed. 
 pip install httpx
 python verify_local_setup.py
6 Claude Code plugin no longer needs an api key because it's slowly eating your graphics card and not one in the data center. Yay!
